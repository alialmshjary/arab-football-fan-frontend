Arab Football API
RESTful API for a social media platform designed for Arab football fans, developed as part of a graduation project using ASP.NET Core and SQL Server.

Overview
This API powers a football-focused social platform where users can share posts, follow matches, interact with live events, chat with other users, and submit match predictions.

Main Features
User registration and authentication
Posts and comments management
User following system
Chat and messaging
Match information and live events
Match predictions
Reports management
RESTful API architecture

# Arab Football API ⚽

A robust RESTful API powering a dedicated social media platform designed for Arab football fans. This project was developed as a graduation project leveraging the power of **ASP.NET Core** and **SQL Server**.

---

## 📌 Project Overview

This API serves as the backbone for a football-centric social network. It enables users to share content, follow live match updates, interact with match events in real-time, join community chat rooms, and submit match score predictions.

---

## 🚀 Key Features

* **User Management & Auth:** Secure registration, JWT-based authentication, and profile customization.
* **Social Feed:** Full CRUD operations for user posts, image uploads, and comments.
* **Following System:** Capabilities to follow/unfollow other fans to personalize the social timeline.
* **Real-time Messaging:** Instant chat messaging and live group chat rooms for active fan discussions.
* **Match Center:** Live match tracking, scheduling, and real-time event updates (goals, cards, substitutions).
* **Match Predictions:** Interactive system allowing users to predict match outcomes and earn points.
* **Moderation & Reports:** Administrative tools to report and manage inappropriate content or behavior.
